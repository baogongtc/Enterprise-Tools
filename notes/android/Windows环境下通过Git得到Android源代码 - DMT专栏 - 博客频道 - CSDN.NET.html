Windows环境下通过Git得到Android源代码

2013-12-24 20:06 6196人阅读 评论(10) 收藏 举报
本文章已收录于：
分类：

版权声明：本文为博主原创文章，未经博主允许不得转载。

Git是一个分布式版本控制／软件配置管理软件，原来是linux内核开发者林纳斯·托瓦兹（Linus Torvalds）为了更好地管理linux内核开发而创立的；由于Android操作系统的核心属于Linux核心的一个分支，所以它也由Git管理。要想理深入学习Android，有必要好好查看它的源代码，由于大多数人都是用Window系统，对于Linux的操作不熟悉，现在不用去安装什么虚拟机了，可以直接在Window下安装Git工具下载Android的源码

首先要安装Git

下载地址：http://download.csdn.net/detail/deng0zhaotai/6759635

可以到code.google下载最新的：http://code.google.com/p/msysgit/downloads/list

http://msysgit.github.io/


安装好Git后，在你需要存放源代码的目录下右键会弹出下面一个菜单


选择：Git Bash后出现以下命令行


在命令中输入：git clone http://android.googlesource.com/platform/frameworks/base.git

按回车会进入下载


1.2G这速度等很久。


到这里Android的代码就下载到了。

进入base目录


输入 ls -l 列出当前所有文件


使用git tag命令查看Android所有的发行版本



这时我们可以根据需要下载各个版本的Android源码

git checkout android-4.4_r1


这样我们就得到了Android4.4的源码。

我们可以在ADT中关联它的源代码



如果你只想下载Launcher的源代码可以执行以下命令

git clone http://android.googlesource.com/platform/packages/apps/Launcher.git


我们需要哪部分的源码可以单独下载，这样会快点，以下是所有代码下载的列表，可以把下表放在一个shell文件中，作批量下载。
