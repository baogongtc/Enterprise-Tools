2.2.1 创建和删除远程branch
Ruby代码  
  1. git push origin head:newbranch_name   
  2. git push origin head:feature/newbranch_name     
  3.   
  4. #删除远程分支，其它开发者要git branch －d －r 分支

Java代码

Java代码
