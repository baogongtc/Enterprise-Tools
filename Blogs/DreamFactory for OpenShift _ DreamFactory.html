DreamFactory for OpenShift



Mobilize your enterprise data with OpenShift!


DreamFactory is an open source mobile backend that provides RESTful services for building modern applications. DreamFactory automatically generates a comprehensive, customizable, and secure REST API for any data source. Data sources include SQL databases, NoSQL document stores, file storage systems, and external web services.

Advanced features include server side scripting and customization, single sign-on with AD, LDAP, and OAuth, record-level access control, and interactive documentation. DreamFactory also provides example applications for all major development environments, including JavaScript, AngularJS, iOS, Android, Titanium, and .Net.
